What is DOM

DOM is a programming interface which is used to control , change , remove or add element on html by javascript. in document developer can do any kind of manifestation of a page . DOM mostly used in Web programming .

Purpose

The purpose of DOM is to add, remove, modify html document in logical way. Using DOM structure tree.

Ways of selecting element

  1. using getElementById
  2. using getElementByClassName
  3. using getElementByTagName
  4. using querySelector
  5. using querySelectors

Event bubble

Event Bubble is a situation which occurs when an event executed . first it triggered on the current element then it goes to top level to the parent unless reach to the document.

Callback Function

Callback Function is a function which is used on eventListener function , the specialty of this function is it has no parameter . it only executes on the declared condition .
